# Tikrar MTI Promo Assets Guidelines 🚀

Dokumen ini berisi panduan dan katalog aset promosi untuk Tikrar MTI Apps by Markaz Tikrar Indonesia (MTI). Gunakan panduan ini untuk menjaga konsistensi brand pada pembuatan poster dan caption di masa mendatang.

## 🎨 Panduan Visual & Desain
Untuk menjaga kualitas dan daya tarik pendaftaran, setiap poster harus mengikuti ketentuan berikut:

1.  **Tema Warna**: Dominan **Putih (Minimalist White)** untuk kesan bersih dan premium. Hijau Emerald (#064E3B) dan Emas digunakan hanya sebagai warna pendukung/aksen.
2.  **Elemen Mockup**: Wajib menampilkan antarmuka (UI) dashboard riil dari aplikasi (misal: *Circular Progress* 85% dan *Timeline* Journey).
3.  **Logo**: Selalu sertakan logo resmi Markaz Tikrar Indonesia di posisi strategis.
4.  **Aset Kreatif**: Gunakan elemen 3D (seperti Al-Qur'an, ikon bintang, atau checkmark emas) untuk menambah daya tarik visual (*High-Conversion Design*).
5.  **Evergreen**: Poster utama tidak perlu mencantumkan tanggal update agar dapat digunakan terus-menerus.

## ✍️ Ketentuan Konten & Istilah
1.  **Terminologi**: Gunakan istilah **"Syahadah"** (bukan "Wisuda") untuk tahap kelulusan di Pekan 14.
2.  **Kontak Resmi**: Setiap media promosi wajib mencantumkan informasi berikut:
